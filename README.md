# 母親目線でやさしく確実に促す

やさしく確実にタスクを促すWebアプリ（3ペインUI）です。

## 起動方法

`index.html` をブラウザで開くか、ローカルサーバーで配信してください。

```bash
# 例: Python
python -m http.server 8080
# http://localhost:8080/mother-prompt-app/
```

## 機能

1. **左ペイン** やること一覧（追加・クリックで選択）。↻ボタンで **Gemini API** による優先度最適化。
2. **中央ペイン** 現在のタスク表示。「できた！」で完了。
3. **右ペイン** 母親アイコン、今日のおすすめ（Geminiで候補取得）、TTSトグル。
4. **画面中央下** 今月のAPI見積＋「タスクをまとめて追加」＋ツール表示切替。
5. **メモリー** 右下の📝でメモリー画面を開く。APIキーもここで保存可能。

---

## APIの適用方法（Gemini APIキー）

このアプリの「優先度の最適化」と「今日のおすすめ」は **Google Gemini API** を使います。次のどちらかでキーを設定してください。

### 方法1：メモリー画面から（おすすめ）

1. 画面右下の **📝** ボタンをクリックしてメモリー画面を開く。
2. **「Gemini APIキー（任意）」** の入力欄に、取得したAPIキーを貼り付ける。
3. そのまま閉じる（自動保存されます）。
4. 以降、左ペインの↻ボタンや右ペインの✦でGeminiが使えます。

### 方法2：APIキーの取得手順

1. [Google AI Studio](https://aistudio.google.com/apikey) にアクセスし、Googleアカウントでログイン。
2. 「Create API key」でキーを発行。
3. 発行されたキーをコピーし、上記「方法1」の入力欄に貼り付けて保存。

### 注意

- APIキーはブラウザのローカルストレージにのみ保存され、サーバーには送りません。
- 利用量に応じて「今月のAPI見積」が画面中央下に表示されます。
